📄 AskMyDocs Widget Example

Embed the AskMyDocs chat widget on any website

Live Widget Demo

How to embed on your site:

1. Copy this script tag to your HTML:

<script src="https://your-api.railway.app/widget.js"></script>

2. Add the widget div where you want it to appear:

<div id="askmydocs-widget" data-source="my-docs"></div>

Widget attributes:

data-source (required): The name of the document to search within. Use "all" to search all documents.

Features:

  • ✓ Real-time chat interface
  • ✓ Source citations with match scores
  • ✓ Responsive design
  • ✓ Cross-origin compatible
  • ✓ No dependencies

API Endpoint Reference

The widget uses the following REST API endpoint:

POST /api/chat

Request:

{ "query": "What is the main topic?", "source_name": "my-docs" }

Response:

{ "answer": "The main topic is...", "sources": [ { "name": "document-name", "type": "pdf", "snippet": "Relevant excerpt from the document...", "score": 92.5 } ] }

🚀 Deployment:

To deploy the API, run:

python -m uvicorn backend.api:app --host 0.0.0.0 --port 8000
